## How to run
1. Create a virtual environment
    This can be done via
    ```
    $ python -m env venv
    ```

2. Run the following to get necessary packages
    ```
    $ pip install requirements.txt
    ```

3.  Run the server
    ```
    $ python manage.py runserver
    ```
3. Visit from your browser
    ```
    http://localhost:8000
    ```
